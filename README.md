# neomyst-cdn

Static image CDN for NeoMyst product gallery.

## Structure
```
products/{stok}/scenes/{color}.jpg
```

## URL pattern
```
https://raw.githubusercontent.com/ufuktasli-debug/neomyst-cdn/main/products/{stok}/scenes/{color}.jpg
```

Used by Qukasoft `product/save` API (catbox.moe replacement — was unreliable).
